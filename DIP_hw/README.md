## UI Implementation  
- A PyQt5-based UI integrates all five tasks.  
- Users can interactively perform each task and visualize the results.

## Dependencies  
- Python 3.8  
- OpenCV  
- Matplotlib
- PyQt5  
- Pillow

## How to Run 
1. Install required dependencies

2. Run the PyQt5 UI
    ``` sh
    python main.py
    ```
